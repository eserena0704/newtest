# Vendor Handoff

This repository is a vendor workspace snapshot for Beauskin website integration work.

## Scope

- Add and configure the AI chatbot integration.
- Generate or maintain sitemap output as required by the vendor workflow.
- Test the integration against the live site and local build.
- Provide a summary of changed files and setup steps when complete.

## Live Site

https://beauskin-luxury.vercel.app

## Notes

- Production deployment access, live environment variables, Stripe settings, Blob storage settings, and admin credentials are not included.
- Do not request secrets in GitHub issues or commits.
- Any required chatbot keys, widget IDs, crawler accounts, or third-party credentials should be managed by the vendor.
- If a final embed-only snippet is enough, provide the exact script/tag and placement instructions.
